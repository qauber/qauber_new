IH Business Pro
=========

IH Business pro is a very powerful and feature rich WordPress theme designed for Businesses and Corporates. With this theme you get access to tons of Social Icons, Layouts, Colors, Featured Sections and an awesome front page.

Getting Started
---------------

How to Install the Theme?

1. Go to WP Admin > Appearance > Themes > Add New.
2. Search for "Hanne".
3. Click on Install and Activate.

Alternatively,

1. Go to WP Admin > Appearance > Themes > Add New > Upload.
2. Upload the .zip file, in which you found this README.txt
3. Click on Install and Activate.

Setting it Up
-------------

This theme is built using Customizer. So to Configure theme settings like Featured Content Area, Sliders, Social Icons, etc from 
	
	Dashboard > Appearance > Customizer
